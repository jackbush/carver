# What We Write About When We Write

This is Carver. You write. It stays. Nobody else sees it.

---

## Formatting

Markdown is the syntax. It gets out of your way.

### Headings

Start a line with `#` for a heading. More hashes, smaller the heading. Like cathedral windows — the higher up, the narrower.

### Lists

Things that accumulate:

1. The first draft
2. The second draft
3. Everything after that

Things that don't need an order:

- The light through the window
- The sound of the keys
- What you were trying to say

### Emphasis

Wrap words in `**asterisks**` for **bold**. Single `*stars*` for *italic*. The syntax dims in the editor so you see the shape of the sentence, not the scaffolding.

### Blockquotes

> "He was a writer who understood that the most important things are always the ones left unwritten."

Use `>` to step back and let someone else speak.

### Code

Inline `code` sits in backticks. Blocks get their own room:

```
what matters
  is what you leave
    in
```

## The Bottom Bar

Three buttons, bottom right. That's the whole thing.

**Upload** — the arrow pointing up. Opens your file system. Pick a `.md` file and it lands here, replacing what's open. Good for continuing work you started somewhere else.

**Download** — the arrow pointing down. Saves what's here to a `.md` file. Give it a name, or keep the one it guesses from your first line.

**Settings** — the gear. Editing theme, reading theme, font size, tint, caret style. Changes take effect immediately. Your choices are remembered.
